# Harry Zhang · 张天译

Static personal website for GitHub Pages.

Deploy this repository as `Tianyi-Zhang-NKU.github.io`, or push these files to the root of an existing GitHub Pages repository.

## Files

- `index.html` - content and semantic structure
- `styles.css` - visual system and responsive layout
- `script.js` - theme toggle, project filters, reveal animation, canvas background
- `assets/avatar.jpg` - local GitHub avatar asset

## Local Preview

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173/`.
